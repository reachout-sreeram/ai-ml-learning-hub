# AI/ML Learning Hub 🚀

Welcome to the **AI/ML Learning Hub**—an interactive web application designed to be your gateway into Data Science, Machine Learning, Deep Learning, and Natural Language Processing.

This hub is designed to help students and developers grasp AI/ML concepts, test their knowledge, and explore comprehensive learning paths.

---

## 🌟 Key Features

1. **Structured Learning Paths**: Step-by-step tracks covering Mathematics, Core ML, Deep Learning, and NLP.
2. **Interactive AI/ML Glossary**: A searchable, categorized dictionary containing 83 core AI/ML terms (e.g. transformers, gradient descent, LoRA) using search inputs and category filters.
3. **AI/ML Knowledge Check**: A functional, interactive quiz module with 20 conceptual questions, real-time grading, score calculations, accuracy metrics, and explanation popups.
4. **Resources Directory**: Curated list of papers, courses, books, and interactive tutorials.

---

## 🛠️ Tech Stack

* **Frontend Framework**: React (v19) + Vite
* **Styling**: Tailwind CSS (v4)
* **UI Components**: Shadcn UI & Radix Primitives
* **Icons**: Lucide React
* **Animations**: Framer Motion

## 🚀 Getting Started

To run this project locally, make sure you have [Node.js](https://nodejs.org/) and [pnpm](https://pnpm.io/) installed.

### 1. Install Dependencies
```bash
pnpm install
```

### 2. Run Development Server
```bash
pnpm dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the application.

### 3. Build for Production
```bash
pnpm build
```
The compiled build output will be stored in the `dist/` directory.
